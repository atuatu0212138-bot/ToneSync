# M1 —— WebGL 双路径管线 + 双视图预览（对应 PRD §9 里程碑 M1）

零构建的原生 ES Modules + WebGL2 单页应用。核心算法模块（`js/color.js`）无 DOM 依赖、
与 `m0/reinhard.py` 数值逐位对齐（node 校验通过）。

## 运行

```bash
# 项目根目录（ES Modules / Worker 需要 http，双击打开 file:// 不行）
python3 web/serve.py            # 开发服务器，禁缓存（BUG-007），默认 8123 端口
# 浏览器打开 http://localhost:8123/web/
```

> 不要用 `python3 -m http.server`：它不发缓存头，代码更新后浏览器可能加载到新旧混杂的模块（BUG-007）。若画布出现"渲染异常"提示或空白，强制刷新（⌘⇧R）。

点「使用示例图体验」即可看到全流程：18 张示例图 + 自动生成的青橙参考图。

## 结构

```
web/
  index.html        布局：顶栏 / 画布区 / 右侧调色面板（可拖宽 296–460px）/ 图标窄栏 52px
  css/style.css     深色 UI（视觉规范与 docs/ToneSync-standalone.html 原型一致）
  js/color.js       色彩数学 + Reinhard 统计 + LUT 烘焙 + 模版定义（独立可复用）
  js/stats-worker.js  Lab 统计量 Worker
  js/gl.js          WebGL2 渲染器：双路径 shader、纹理/LUT 管理
  js/app.js         状态与交互编排（单一 GL canvas 承载单图/墙/胶片条）
```

## 测试参数（自动化/调试用）

`?demo=1` 自动载入示例图 + 参考图；`&mode=wall` 直接进墙视图；`&preset=2` 激活第 3 个模版；
`&strength=40` 设强度；`&selftest=1` 显示色彩数学自检（对照 Python 参考值）；`&debug=1` 显示状态浮层。
